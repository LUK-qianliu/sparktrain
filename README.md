# sparktrain
这是一个spark流处理慕课网的课程日志项目,流程图如下：
![架构图](https://img-blog.csdnimg.cn/2019042717023825.png)

本项目只是实现了
1.数据采集（flume），
2.对接消息队列(kafka),
3.sparkstreaming处理数据流
4.保存数据到hbase

还有一个springboot+echarts可视化是后半部分的项目，仓库在https://github.com/LUK-qianliu/springboot1  。
